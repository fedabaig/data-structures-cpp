# Data Structures in C++

This repository contains my C++ implementations of core data structures, algorithms, and applied programming projects developed through coursework and hands-on practice. The projects focus on problem solving, structured data management, algorithm design, and real-world style applications.

## Topics Covered

- Linked Lists
- Ordered Linked Lists
- Stacks
- Queues and Priority Queues
- Binary Search Trees
- AVL Trees
- Hash Tables
- Graphs
- Merge Sort
- Backtracking
- Input Validation and Parsing
- Object-Oriented Programming (OOP)

## Projects Included

### Network Routing Graph Pathfinding
A graph-based project that reads network routes from a file and finds a path between IP addresses.

### Bug Tracking System with Priority Queue
A bug tracking simulation that prioritizes bug reports using a priority queue and manages workflow states.

### Sudoku Solver Using Backtracking
A Sudoku solver that loads a puzzle from a file and solves it using a backtracking algorithm.

### Browser Navigation Simulation Using Stacks
A browser history simulation that supports back and forward navigation using stacks.

### Hash Table Collision Resolution
Implementations of hash tables using:
- Open addressing with quadratic probing
- Separate chaining with linked lists

### Product Catalog Binary Search Tree
A product management system that stores product records in a binary search tree and supports search, deletion, and in-order traversal.

### Merge Sort Product Listings
A project that reads product data from a file and sorts linked lists by description, price, and rating using merge sort with function pointers.

### Burger Builder Using AVL Tree
A burger customization system that stores toppings in a custom set backed by an AVL tree.

### Ordered Linked List with Custom Objects
A project that stores snack objects in an ordered linked list and maintains sorted order automatically.

### YarnSkein OOP and Enum Mapping
A class-based project that models yarn data using enums, static maps, validation, and object-oriented design.

### Time Processing and Validation
A time parsing and validation project that uses regular expressions, exception handling, and object-oriented design to process time input.
